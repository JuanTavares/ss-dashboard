Home

Este projeto é a apresentação de uma simples dashboard em Angular 6, contendo um total de 5 telas (contando com a Home) e utilizando diferentes recursos.

A tela de Login possui validação de formulário, impedindo assim que o usuário clique no botão de logar caso o formulário esteja inválido.

Após efetuar o login, o usuário é encaminhado para a Dashboard, que possui um header e uma side-bar com as opções de rotas dentro da aplicação.

A primeira tela da Dashboard é uma tela onde são exibidos alguns gráficos com dados fake usados apenas para dar o formato de cada um.

A tela de Usuários exibe uma lista de 101 usuários recebidos da API "https://randomuser.me", e cada usuário, quando clicado, exibe uma tabela de dados do lado.

A tela do Editor de Conteúdo exibe um editor semelhante ao utilizado pelo Wordpress, sendo esse gerado através da ferramenta open-source TinyMCE. Atenção: O editor em si pode ser usado apenas para escrever e testar suas capacidades, mas não é funcional para edição de qualquer conteúdo dentro desta aplicação.

A tela de Lista de Interesses exibe uma tabela que faz agrupamento de pessoas por cargo de trabalho e outra tabela que agrupa por área de interesses. Devido a limitação da API "https://randomuser.me", eu criei uma pequena API através do site "https://randomapi.com" sugerido pelos mesmos, porém essa api só pode exibir grupos de 10 usuários de cada vez e tem limite de 500 gerações aleatórias por dia.

A aplicação foi estilizada com Bootstrap e com algumas classes em css customizado.
